# Contributing to LoopZero

We welcome developers, researchers, psychologists, and AI engineers to collaborate. Here’s how:

- Fork the repo
- Create a feature branch
- Submit a PR with detailed context

## 📘 Documentation Help
- Improve READMEs, whitepaper drafts
- Translate project content

## 🧪 Behavior Model Training
- Submit anonymized behavior loop samples
- Help train our neural interruption dataset
